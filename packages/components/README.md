# components

## Usage
Add module using 
```
yarn add @molgenis-ui/components
```
Then in your vue app 
```
import { [MyComponent] } from '@molgenis-ui/components'
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
